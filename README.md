# Mini SOC Detection & Incident Response Lab

## Project Overview
This project demonstrates a hands-on SOC-style lab focused on SIEM-based alert detection, log analysis, and incident response across Windows and Linux endpoints.

## Lab Architecture
- Splunk SIEM (Manager)
- Windows endpoint (log source)
- Linux endpoint (log source)
- Attacker machine (optional)


## Attack Scenarios
- Brute-force authentication attempts
- Suspicious PowerShell or script execution
- Malicious file execution

## Detection Goals
- Detect and investigate authentication abuse
- Identify suspicious process execution
- Analyse and respond to malicious activity
